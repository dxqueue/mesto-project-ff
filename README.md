# Проектная работа Mesto

Одностраничный сайт со следующим функционалом:
- Профиль:
  - Изменение аватара профиля
  - Изменение описания и имени профиля
- Карточки:
  - Просмотр карточек
  - Добавление и удаление собственных каточек
  - Реакции на карточки
